# esports
 NLW esports 

## Back-end
- Nodejs
- Type Script
- SQLite

## Front-end
- React
- React Native

## Funcionalidades da WEB
- Vizualizar anúncios de games.
- Cadastrar Duo para determinado game.

## Funcionalidades do Mobile
- Vizualizar anúncios de games.
- Selecionar game e escolher um Duo para jogar através do usuário do discord.
